# 📌 Pinterest Clone

A full-stack Pinterest-style web application built using **React (Vite)**, **Node.js**, **Express**, and **MongoDB**.  
It allows users to create pins, upload images, follow users, comment on posts, and explore a social feed.

---

## 🚀 Live Demo

- 🌐 Frontend: https://pinterest-clone-inky.vercel.app  
- ⚙️ Backend: https://pinterest-clone-y6av.onrender.com  

---

## ✨ Features

### 🔐 Authentication
- User register & login
- JWT authentication
- HTTP-only cookies for security

### 📌 Pins
- Create pins with images
- View all pins feed
- View single pin details
- Edit & delete own pins

### 💬 Social Features
- Follow / unfollow users
- Comment on pins
- View followers & following count

### 🖼 Image Uploads
- Cloudinary integration for image storage

### 📱 UI
- Fully responsive design (mobile + desktop)
- Pinterest-like UI layout

---

## 🛠 Tech Stack

**Frontend:**
- React (Vite)
- React Router DOM
- Context API
- Axios
- Tailwind CSS

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt
- cookie-parser
- CORS

**Services:**
- Cloudinary (image hosting)
- Vercel (frontend deployment)
- Render (backend deployment)
